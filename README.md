# Aetheric Chronicle | Cinematic Digital Memoir

A high-end, immersive digital portfolio built with **React**, **Vite**, and **Framer Motion**. This project transforms the traditional portfolio into a cinematic chat-driven experience, serving as a digital representation of a professional journey.

## 🌟 Overview

**Aetheric Chronicle** is more than just a resume; it's a narrative-driven platform where visitors can engage with an AI agent to explore the background, skills, and projects of **Abir**, a Software Solutions Consultant.

Built with a focus on **Premium Aesthetics** and **Seamless Interactions**, it uses:
- **Glassmorphism & Dark Mode**: A sleek, modern UI designed for high-end professional presentation.
- **Micro-animations**: Powered by Framer Motion for a fluid, "alive" interface.
- **Conversational UI**: A terminal-inspired chat interface that provides context-aware information.

## 🚀 Tech Stack

- **Core**: React 18 + Vite
- **Styling**: Vanilla CSS (Modern CSS variables, HSL color palettes, Custom noise overlays)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Typography**: Instrument Serif & Inter (via Google Fonts)

## 🛠️ Features

- **Interactive Chat Agent**: Ask about experience, skills, or projects.
- **Chapter-based Narrative**: Structured data mapping to professional milestones.
- **Responsive Design**: Fully optimized for Desktop and Mobile viewports.
- **Performance Optimized**: Lightning-fast loads via Vite's build pipeline.

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shahriarhaqueabir/HawkwardWebsite.git
   cd HawkwardWebsite
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `aetheric-chronicle.jsx`: The main entry point for the immersive chat experience.
- `main.jsx`: React root initialization.
- `public/`: Static assets, including the custom `favicon.svg`.
- `index.html`: Main entry HTML with font injections and metadata.
- `vite.config.js`: Build and plugin configuration.
